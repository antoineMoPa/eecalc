eecalc

Github project | backend: math.js | syntax documentation: mathjs docs

Usage:
Backspace to delete an empty cell
Use semicolon (;) to write what you would write on many cells:
a = 1; b=2; sin(a+b)
Numbers:
You can use the electrical engineering notation with numbers (n,u,m,k/K,meg/M,G)
ex: 1K = 1000

Math:
sin(3.14)
sin(360deg)
a = 30
b = 3 * a
b

Conversions:
3 m to inches
3 in to cm
1 gal to L
1 L to cm^3

Constants:
pi, e (euler), i (sqrt(-1)), etc. mathjs constants

Bonus:
LL(a,b,...) : value of resistances in parallel.
